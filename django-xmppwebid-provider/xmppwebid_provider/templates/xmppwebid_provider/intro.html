{% blocktrans %} xmppwebid certificate generator is an Django app intended to generate valid certificates for HTPP and XMPP WebID authentication. {% endblocktrans %}

{% blocktrans %} It creates XMPP account and generates the certificate that is installed in your browser and can also be installed in a XMPP client. {% endblocktrans %}

{% blocktrans %} To use your certificate in XMPP (Jabber )you would need a client that supports certificates (currently Gajim v0.14) and a Jabber server software that supports SASL EXTERNAL authentication (Jabberd). {% endblocktrans %}

{% blocktrans %} For more information about this module and enabled Jabber servers and clients, see the xmppwebid project site. {% endblocktrans %}
